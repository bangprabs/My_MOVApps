# MOV Apps
Project pemesanan tiket bioskop secara online berbasis mobile (Android). Dengan kenunggulan yaitu Pre Sale dan Discount tiket bioskop untuk pembelian tiket tertentu

## Feature
1. Splash Screen
2. Onboarding 
3. Sign In & Sign Up
4. E - Wallet
5. QR Code

## Technology
Firebase

## UI / UX
Using Adobe XD

## Source Code
Using kotlin for programming language
